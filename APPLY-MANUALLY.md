## How to Use This List

1. When you have time, open the Greenhouse portal links and apply through their web forms
2. After applying via web form, update this file with `Status: Applied on [DATE]`
3. Log the application to `applications.json` with `appliedVia: "greenhouse"` or `appliedVia: "manual"`
4. Commit with: `git commit -m "apply: Bitpanda + ConsenSys manual applications"`

## Future Sessions

- Is this company in `APPLY-MANUALLY.md`?
