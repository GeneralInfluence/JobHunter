### Structure of Each Entry

Each application or project pursuit is a record with the following fields:

| Field | Description |
|---|---|
| `id` | Unique identifier for this entry |
| `company` | Name of the employer or client |
| `role` | The position or project Sean applied for or proposed |
| `appliedDate` | Date the application or proposal was submitted (ISO format: YYYY-MM-DD) |
| `appliedVia` | How Sean applied: `email`, `ashby` (job portal), `referral`, or `raidguild` (consulting/freelance) |
| `status` | Current status of this pursuit (see below) |
| `notes` | Additional detail — responses received, follow-ups, outcome |
| `contact` | Key contact person at the company (where applicable) |
| `url` | Link to the job posting or project (where available) |

### Status Definitions

| Status | Meaning |
|---|---|
| `pending` | Application submitted, awaiting response |
| `active` | Actively in discussions or negotiation |
| `active-client` | Engagement underway; client has paid or committed |
| `nurturing` | Ongoing relationship; awaiting their next step |
| `warm` | Positive contact made; no formal proposal yet |
| `pending-follow-up` | Proposal made; following up |
| `stalled` | Communication has gone quiet; may re-engage |
| `cold` | No response after outreach |
| `rejected` | Formal rejection received |
| `closed` | Opportunity ended without placement |
